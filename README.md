# Case Simulator 1

Консольный симулятор открытия кейсов с анимацией и системой крафта.

## Запуск

```powershell
python main.py
```

## Структура проекта

```
case_simulator/
├── models/          # Базовые классы: Item, Case, Inventory
├── scenes/          # Сцены игры: меню, открытие кейсов, крафт
├── utils/           # Утилиты для работы с консолью
├── data/            # Примеры предметов и кейсов
├── state.py         # GameState с балансом и инвентарем
└── app.py           # Главный класс приложения
```

## Функциональность

- **Баланс (депозит)**: Отображается в меню и пополняется продажей предметов
- **Открытие кейсов**: Анимация рулетки с замедлением, выбор оставить или продать предмет за 80%
- **Инвентарь**: Хранит кейсы и предметы игрока

## Добавление контента

Примеры предметов и кейсов находятся в `case_simulator/data/presets.py`.

Создайте новый предмет:
```python
NEW_ITEM = Item(id="item_id", name="Item Name", price=500, category="weapon", rare=1)
```

Создайте новый кейс:
```python
NEW_CASE = Case(
    id="case_id",

    
    name="Case Name",
    price=200,
    items=(ITEM1, ITEM2, ITEM3),
)
```
